# Todo App

一个轻量级网页待办事项系统，支持普通用户和管理员两种角色。

## 功能

- 用户注册/登录
- 任务创建、编辑、删除、状态管理
- 任务分配与协作
- 管理员面板（用户管理、统计）

## 技术栈

- 后端：Python Flask + SQLite
- 前端：HTML/CSS/JavaScript

## 运行

```bash
pip install flask
python app.py
```

## 开发过程

本项目通过模拟真实协作开发流程完成，共经历 5 轮迭代：

| 轮次 | 类型 | 改动 |
|------|------|------|
| 第1轮 | feat | 初始版本 |
| 第2轮 | refactor | 代码质量优化 |
| 第3轮 | feat | 用户体验优化 |
| 第4轮 | feat | 功能增强 |
| 第5轮 | fix | Bug 修复 |
